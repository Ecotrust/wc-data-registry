Connect

This portal provides several pathways for data users and application developers to search and connect to the datasets that they discover.

CSW - Catalog Services for the Web

Access our CSW at:

http://portal.westcoastoceans.org/geoportal/csw

For example, the following URL queries the capabilities of our CSW:

http://portal.westcoastoceans.org/geoportal/csw?service=CSW&version=2.0.2&request=GetCapabilities

ESRI Geoportal REST API

Access our Geoportal REST API at:

http://portal.westcoastoceans.org/geoportal/rest/

For example, the following URL queries all available metadata records in the portal and returns it in the JSON format:

http://portal.westcoastoceans.org/geoportal/rest/find/document?f=pjson

Geoportal REST API Documentation


SOLR REST API

Access our SOLR REST API at:

http://portal.westcoastoceans.org/solr/collection1/select

For example, the following URL selects the first 20 metadata records for the marine debris category as JSON:

http://portal.westcoastoceans.org/solr/collection1/select?&q=*+sys.src.collections_ss:%20(Issue|Marine_Debris)&rows=20&wt=json
